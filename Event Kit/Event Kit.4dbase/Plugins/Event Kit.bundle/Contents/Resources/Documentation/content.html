Event Kit

Plugin Identifier: 20000

Summary

This plugin allows 4D to access data used by Calendar.app and Reminders.app on Mac.

The Event Kit framework is used to access the Calendar database, which holds both calendar and reminder information. The API (and therefore the plugin) requires Mac OS X 10.8 or later.

The Event Kit framework is available for 64-bit applications only. 4D is 32-bits, which means it can't take advantage of this framework. (we can't mix 64-bits and 32-bits in the same thread.) To overcome this problem, the plugin launches a faceless Cocoa application called EventKitHelper. It then uses the NSConnection class to establish a connection between 4D (which is 32-bits) and the helper application (which is 64-bits). This connection is transparent to the 4D code using the plugin.

EK Store

EK STORE Commit

success:=EK STORE Commit

success (LONGINT)

Discussion

Commits changes to the Calendar database. Changes made to a calendar, event or reminder are not stored in the database until you actually commit them.
